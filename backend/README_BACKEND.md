# Backend Development Environment – Coach Xai

This directory contains optional scripts and configuration files used to test and
validate Coach Xai logic locally before integration with Cada Menudo's backend.

## Files
- **xai_dev.py** – Local prototype runner.
- **config.example.json** – Example config template (no real keys).

## Purpose
- Validate module responses.
- Test the core logic prompt.
- Run evaluation cases in `/tests/cases/`.
